# AssetMind AI

AI-powered Industrial Knowledge Intelligence platform.

## Goal

Convert scattered industrial documents into an asset-centric operations brain using document ingestion, RAG, knowledge graphs, RCA, and compliance intelligence.

## Structure

- `apps/` - frontend and backend apps
- `docs/` - architecture, planning, demo notes
- `sample-data/` - synthetic industrial sample data
- `infra/` - deployment and infrastructure notes
